## Quick local testing
1. Start a simple local server from the project root (macOS / zsh):

```bash
python3 -m http.server 8000 --directory "/Users/sachin/Study Material/Portfolio/Sachin"
```

2. Open the site in your browser: http://localhost:8000

3. If changes to images or CSS don't appear immediately, force reload or clear cache:
    - Hard refresh: Cmd+Shift+R (macOS) or Ctrl+F5 (Windows).
    - Or increment the cache-bust query string in the file (e.g. `?v=3`) and reload.

## Notes & tips
- If you plan to publish via GitHub Pages, push your changes and ensure the repository is configured for Pages. Renaming image files (instead of using query parameters) is a good way to make cache-busting permanent for visitors.
- I kept a fallback background color in `assets/css/style.css` so the page still looks okay if the image fails to load.
